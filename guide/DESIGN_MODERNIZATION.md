## 디자인 방향: Ethereal Glass

### Vibe 선택 이유

개발 문서 사이트에 어울리는 **프리미엄 다크 테마 + 글래스 효과**:
- 깊은 네이비 배경 (`#0c1222`) — 순수 검정 대신 색감 있는 어두운 배경
- 글래스 카드 — `backdrop-blur-20px` + `border-white/8` + 내부 빛 반사
- 밝은 텍스트 대비 — `#f0f4fc` (기존 `#f1f5f9`보다 약간 밝게)
- 더 넓은 간격 — 섹션 간 `py-24`, 요소 간 `mb-8`
- 더 큰 Mermaid — `min-height: 300px`, 패딩 확대

### 색상 팔레트 비교

| 요소 | 기존 | 변경 후 | 이유 |
|------|------|---------|------|
| 배경 | `#0a0e1a` (거의 검정) | `#0c1222` (깊은 네이비) | 색감 추가 |
| 카드 | `#1a2035` (어두운 회색) | `rgba(255,255,255,0.04)` (글래스) | 투명감으로 깊이 |
| 보더 | `#2a3454` (진한 남색) | `rgba(255,255,255,0.08)` (반투명) | 부드러운 구분 |
| 본문 텍스트 | `#94a3b8` (회색) | `#a8b8d8` (밝은 회색-블루) | 가독성 향상 |
| 보조 텍스트 | `#64748b` (어두운 회색) | `#6b7fa3` (밝은 블루그레이) | 대비 강화 |
| Accent Blue | `#3b82f6` | `#4d8df5` | 약간 밝게 |
| Accent Purple | `#8b5cf6` | `#9d7af5` | 약간 밝게 |

## 변경 대상 파일 및 수정 내용

### 1. `globals.css` — 색상 팔레트 + 타이포그래피 + 효과

```
변경:
- :root 색상 변수 전면 교체 (위 팔레트)
- body line-height: 1.7 → 1.8 (더 넓은 줄간격)
- word-break: keep-all (한글 단어 단위 줄바꿈)
- h1~h4에 letter-spacing: -0.02em (한글 헤딩 시각 밀도 감소)
- p, li에 text-wrap: pretty (더 나은 줄 분배)
- .glass-card 클래스 추가 (글래스모피즘 기본 스타일)
- grain texture overlay (미묘한 노이즈로 프리미엄 질감)
- Mermaid svg min-height: 200px → 300px
- @media (prefers-reduced-motion) 추가 (접근성)
- 구문 강조 색상 약간 밝게 (가독성)
```
